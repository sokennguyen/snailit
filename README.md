# SnailNest

A simple React + Vite + Tailwind + daisyUI project.

## Features
- Mobile-first UI
- Tailwind CSS & daisyUI
- React 19

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```

## Project Structure
- `src/` — Main source code
- `public/` — Static assets
